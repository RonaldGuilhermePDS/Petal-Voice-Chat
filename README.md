## Getting Started

Follow the instructions to start the project without any problems.

## Installation

1. Clone the Repository
```sh
git clone https://github.com/RonaldGuilhermePDS/Petal-Voice-Chat.git
```
2. Install Packages
```sh
mix deps.get
```
3. Create and migrate your database
```sh
mix ecto.setup
```
4. Run
```sh
mix phx.server
```
